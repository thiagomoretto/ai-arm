# Exemplo de um artigo técnico/científico utilizando Latex.

Esta pasta contém os arquivos necessários para compilar o texto "artigo.tex"
utilizando o Latex. É preciso, no entanto, verificar antes se os pacotes
"algorithm", "vaucanson" e "sbc-template" estão instalados no seu sistema.
Uma versão em pdf do artigo também está disponível "artigo.pdf". Para gerar
novamente essa versão basta utilizar, em um shell do linux, os comandos:

$ make clean
$ make
